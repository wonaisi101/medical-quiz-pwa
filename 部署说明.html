📱 让手机完全脱离 PC

以下方法任选一种，部署后手机离线永久可用：

方法一：GitHub Pages ⭐ 推荐

免费 · 永久 · 无需信用卡

1 注册 GitHub 免费账号（已有则忽略）
2 登录后点右上角 + → New repository
取名 medical-quiz → 选 Public → 点 Create
3 进入新仓库 → 点 uploading an existing file
打开电脑的 MedicalQuizPWA/dist/ 文件夹
全选所有文件 + 文件夹 → 拖入浏览器上传
4 点 Commit changes → 稍等几秒
5 点顶部 Settings → 左侧 Pages
Branch 选 main → 点 Save
等 1-2 分钟出现地址： https://你的用户名.github.io/medical-quiz
6 手机 Safari 打开这个地址 → 分享按钮 → 添加到主屏幕
✅ 以后无需电脑，开飞行模式也能用！

后续更新：重新把新 dist/ 文件上传到同一个 GitHub 仓库覆盖即可

方法二：iPhone 本地服务器（不依赖任何网站）

1 App Store 搜索安装 "Documents by Readdle"（免费）
2 打开 Documents → 点底部 连接 → 开启 Wi-Fi 传输
电脑浏览器打开显示的地址 → 把 dist/ 整个文件夹上传
3 回到 Documents → 找到上传的文件夹
点底部 ··· → 选择 Web Server → 启动
4 iPhone 自带 Safari 打开 http://localhost:xxxx
→ 分享按钮 → 添加到主屏幕
✅ 以后无需电脑无需网络，在 Documents 里点"Web Server"启动即可

dist/ 文件夹位置: e:\我的文件\全科医学刷题系统\001\MedicalQuizPWA\dist
